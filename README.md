# TorqueDesk AI — Website Design

A premium, AI-powered auto shop management platform website built with **Next.js 14** and **TypeScript**.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
